Zougloub-overlay

This is the index of gentoo ebuilds we host.

This overlay can be accessed at :

  • rsync://gentoo.zapto.org/portage-trees/zougloub
  • git://git.zougloub.eu/zougloub-overlay.git
  • recursive http download with 
    wget -nv -r -np --reject "index.html,robots.txt" --no-host-directories "http://gentoo.zapto.org/repos/zougloub" --directory-prefix="/usr/portage"

You can add http://Gentoo.zapto.org/layman.xml to your layman overlays list to add it simply.
